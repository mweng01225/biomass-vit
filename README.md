\# Biomass Prediction with Vision Transformer (ViT-B/16), built for a kaggle competition.



\## Overview

This project predicts biomass from image data using a pretrained Vision Transformer (ViT-B/16). The goal was to improve regression performance on a small real-world dataset.



\## Results

\- Improved R² from -0.51 → 0.49

\- Replaced CNN baseline with ViT

\- Tuned training pipeline and preprocessing



\## Tech Stack

\- PyTorch

\- Hugging Face Transformers

\- Vision Transformer (ViT-B/16)



\## Key Features

\- Custom dataset + dataloaders

\- Image preprocessing + augmentation

\- Regression training pipeline
